# Types de structures linéaires

Les structures linéaires permettent d’organiser les données de manière séquentielle. Les principales structures linéaires sont :

- **Tableau** : structure statique avec accès direct par index.
- **Liste chaînée simple** : chaque élément pointe vers le suivant.
- **Liste chaînée double** : chaque élément pointe vers le précédent et le suivant.

Chacune de ces structures présente des avantages et des inconvénients selon les opérations à effectuer (ajout, suppression, recherche, etc.).
